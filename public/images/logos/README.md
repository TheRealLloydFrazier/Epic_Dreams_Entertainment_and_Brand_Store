# Logo Assets

This directory contains logo files for the Epic Dreams Organization.

## File Naming Convention

Each company has three logo variants:

| Company | Primary | Hero | Embroidery |
|---------|---------|------|------------|
| Holding Company | `holding-primary.png` | `holding-hero.png` | `holding-embroidery.png` |
| Entertainment | `entertainment-primary.png` | `entertainment-hero.png` | `entertainment-embroidery.png` |
| Energy | `energy-primary.png` | `energy-hero.png` | `energy-embroidery.png` |
| AI Solutions | `ai-primary.png` | `ai-hero.png` | `ai-embroidery.png` |

## Logo Variants

### Primary Mark
- **Use:** Navigation, product tags, labels
- **Style:** Simple, clean infinity-in-circle
- **Recommended size:** 200x200px minimum
- **Format:** PNG with transparency

### Hero Emblem
- **Use:** Marketing assets, site hero sections
- **Style:** Detailed 3D cosmic/galaxy effect
- **Recommended size:** 500x500px minimum
- **Format:** PNG with transparency

### Embroidery Mark
- **Use:** Merchandise display (hats, polos, clothing)
- **Style:** Simple 1-2 color, optimized for embroidery
- **Recommended size:** 200x200px minimum
- **Format:** PNG with transparency

## Export from Figma

1. Select the logo frame in Figma
2. Export as PNG at 2x or 3x scale
3. Ensure "Include background" is unchecked for transparency
4. Save with the appropriate filename from the table above
